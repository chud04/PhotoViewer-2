Requirements

  • API from Flickr
  • Widgets used include - HorizontalPanel, TextBox, ListBox, and Button.
  • Functionality:

    1. Entering a username into the TextBox and hitting the submit button requests publicly viewable photo-set information for that user. It sets the output div to loading... while waiting for AJAX to return.
    The returned photo-sets populate the ListBox with the photo-set names. It also displays the set count in the output div.

    2. Selecting a photo-set in the ListBox will send a request to the server for the photos in the selected photo-set. It sets the output div to loading... while waiting for AJAX.
    The returned photos populate the output div.

    3. I added history management to the page also. This allows the user to use the back & forward buttons with AJAX requests. The data is not cached but the events fired are stored as parameters. This also allows for the sending of links via email/twitter/etc.

    4. I also added some exception handling. I did this in two ways. One is server side by using the PhotoViewerException class, the other is client side by reading the JSON stat in the response.

  • The server processes XML in order to find the userid of the username searched for. The server then uses the Flickr JSONP API to populate the FlickrUser attributes as strings. The client will evaluate the JSONP callbacks.
  • I modified the CSS to keep it centered.
